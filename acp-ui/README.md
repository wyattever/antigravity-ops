# Antigravity Control Plane (ACP)

The Antigravity Control Plane (ACP) is a sovereign infrastructure UI extension for VS Code and compatible forks. It serves as a local agentic orchestrator, establishing visibility and control over autonomous AI agents like SYNAPSE.

## Features
- **Sovereign Sidebar**: Provides a decoupled, dedicated view for agent observability.
- **Status Checkpoint (SUP Protocol)**: A manual override to pause the agent and request a state export.
- **Heartbeat Monitoring**: Real-time health checks of the agentic control plane.
- **Panic Rollback**: Instantly revert to a previous ACP Snapshot.

This repository serves as the public face of our Sovereign infrastructure claiming the Open VSX namespace for `antigravity-ops`.
